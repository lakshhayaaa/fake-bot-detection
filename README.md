# fake-bot-detection

## Project set-up
### 1. Create virtual environment
```bash
python -m venv myenv
```
### 2. Install Requirements
```bash
pip install requirements.txt
```

## Instruction to run

### 1. Data Preprocessing
```bash
python app/services/preprocessing.py
```
The preprocessed data is saved in "data\processed\cleaned_data.csv"
